# Breadcrumb Component

Navigation trail that shows users their current location within the site hierarchy and provides easy backtracking.

## Usage Guidelines

| DO | NEVER |
|:---|:------|
| Show clear hierarchy from general to specific | Use breadcrumbs for single-level navigation |
| Make intermediate levels clickable when possible | Make the current page clickable |
| Use consistent separators (> or /) | Mix different separator styles |
| Limit to 5-7 levels to prevent cognitive overload | Show identical information as main navigation |

## Accessibility Features

- **WCAG AAA Compliance:** Proper landmark navigation and screen reader support
- **Keyboard Navigation:** Tab and Enter key support for interactive elements
- **ARIA Support:** Breadcrumb navigation landmark with current page indication
- **Structure:** Semantic list markup with proper heading hierarchy

## Navigation Patterns

| Depth Level | Recommended Pattern | Cognitive Load |
|-------------|-------------------|----------------|
| 2-3 levels | Full breadcrumb | 1/10 |
| 4-5 levels | Standard display | 2/10 |
| 6+ levels | Collapsed middle | 3/10 |
