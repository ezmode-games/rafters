# Tooltip Component

Contextual help overlay that provides additional information without interrupting user workflow.

<ComponentPreview 
  component="tooltip" 
  showIntelligence={true}
  height="200px"
  client:load
/>

## Usage Guidelines

| DO | NEVER |
|:---|:------|
| Keep tooltip content concise (1-2 sentences) | Critical information only in tooltips |
| Use for explaining unfamiliar icons or terminology | Tooltips on mobile without click/tap trigger |
| Provide helpful context without duplicating visible text | Overlapping tooltips that create visual chaos |
| Ensure tooltips work with keyboard navigation | Long-form content better suited for modals |

## Accessibility Features

- **WCAG AAA Compliance:** Keyboard accessible and screen reader compatible
- **Focus Management:** Proper ARIA labeling and role assignments  
- **Dismissal Patterns:** Escape key and click-outside behavior
- **Timing Controls:** Persistent display options for users who need more time

## Content Patterns

| Content Type | Recommended Length | Use Case |
|--------------|-------------------|-----------|
| Icon explanation | 2-5 words | "Save changes" |
| Help text | 1-2 sentences | Form field guidance |
| Status info | 3-8 words | "Last updated 2 hours ago" |
| Keyboard shortcut | Combination | "Ctrl+S" |
