## Usage Guidelines

| DO | NEVER |
|:---|:------|
| Group related chips with consistent spacing | Long text that breaks visual integrity |
| Use clear, concise labels (1-3 words) | Critical functionality solely in removable chips |
| Provide visual feedback for interactive states | Mixed interaction patterns within same group |
| Limit chip groups to 7Â±2 items when possible | Chips for primary navigation or major actions |

## Accessibility Features

- **WCAG AAA Compliance:** Keyboard accessible with proper focus indicators
- **Screen Reader Support:** Clear labels and state announcements
- **Touch Targets:** 44px minimum for interactive elements
- **Keyboard Navigation:** Enter/Space for selection, Delete for removal
