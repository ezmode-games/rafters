## Usage Guidelines

| DO | NEVER |
|:---|:------|
| Limit navigation items to 5-9 primary sections | Create sidebars that compete with main content |
| Use consistent visual hierarchy for nested items | Nest navigation more than 3 levels deep |
| Provide clear active state indicators | Use sidebars for primary content on mobile |
| Ensure responsive collapse behavior | Mix different interaction patterns within same sidebar |

## Accessibility Features

- **WCAG AAA Compliance:** Proper landmark navigation and heading structure
- **Keyboard Navigation:** Tab order and arrow key support for nested items
- **Screen Reader Support:** Clear navigation landmarks and skip links
- **Focus Management:** Proper focus trapping when collapsed/expanded
