## Accessibility

- Uses native `<button>` element by default
- `aria-disabled` and `aria-busy` for loading/disabled states
- Keyboard activation with Enter and Space
- Focus visible ring for keyboard navigation
- When using `asChild` with non-button elements, adds `role="button"` and keyboard handlers
