## Accessibility

- **Focus Trapping**: Focus is trapped within the dialog when open
- **Escape Key**: Closes the dialog by default
- **Backdrop Click**: Closes the dialog by default (configurable)
- **ARIA Attributes**: Proper `role="dialog"`, `aria-modal`, `aria-labelledby`, `aria-describedby`
- **Body Scroll Lock**: Prevents background scrolling when modal is open
