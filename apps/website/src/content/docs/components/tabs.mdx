## Keyboard Navigation

| Key | Action |
|-----|--------|
| `Tab` | Focus the active tab trigger |
| `ArrowLeft` | Move to previous tab |
| `ArrowRight` | Move to next tab |
| `Home` | Move to first tab |
| `End` | Move to last tab |

## Accessibility

- Uses proper ARIA `tablist`, `tab`, and `tabpanel` roles
- Automatic `aria-selected` and `aria-controls` attributes
- Roving tabindex for keyboard navigation
- Focus rings for keyboard users
