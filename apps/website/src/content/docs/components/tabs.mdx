# Tabs Component

Content organization component that manages information architecture through contextual grouping.

<ComponentPreview 
  component="tabs" 
  showIntelligence={true}
  height="300px"
  client:load
/>

## Usage Guidelines

### DO

- Limit tab count to 5-7 items for optimal cognitive load
- Use clear, descriptive tab labels
- Keep tab content contextually related
- Provide visual indicators for active state

### NEVER

- Nest tabs within tabs (compounds complexity)
- Long tab labels that break visual layout
- Critical actions hidden in inactive tabs
- Tabs for sequential processes (use steps instead)

## Accessibility Features

- **WCAG AAA Compliance:** Keyboard navigation with arrow keys and Tab
- **Focus Management:** Proper focus indicators and tab sequence
- **Screen Reader Support:** ARIA tabs pattern with role and state announcements
- **Keyboard Shortcuts:** Arrow keys for tab navigation, Enter/Space for activation
