## Semantic Usage

| Status | Variant | Example Use |
|--------|---------|-------------|
| Active/Live | `success` | Published, enabled, online |
| Pending | `warning` | Review needed, processing |
| Inactive | `default` | Draft, disabled, archived |
| Error/Failed | `destructive` | Failed, blocked, invalid |
| Informational | `info` | Tips, new features |
