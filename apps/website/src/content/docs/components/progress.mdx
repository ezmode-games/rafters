# Progress Component

Visual progress indicator that manages user expectations and reduces anxiety during loading or multi-step processes.

<ComponentPreview 
  component="progress" 
  showIntelligence={true}
  height="200px"
  client:load
/>

## Usage Guidelines

| DO | NEVER |
|:---|:------|
| Provide progress indicators for operations over 2 seconds | Use progress bars for instant operations |
| Use determinate progress when duration is known | Create anxiety with overly precise timing |
| Include contextual labels and time estimates | Hide progress for critical user operations |
| Show completion states with positive feedback | Use indeterminate progress for known durations |

## Accessibility Features

- **WCAG AAA Compliance:** Proper role and aria-valuenow attributes
- **Screen Reader Support:** Progress announcements and status updates
- **Keyboard Navigation:** Focusable when interactive
- **Visual Indicators:** High contrast and clear completion states

## Progress Patterns

| Operation Type | Progress Pattern | Trust Building |
|----------------|------------------|----------------|
| File upload | Linear with % | Show exact progress |
| Form validation | Stepped | Clear completion |
| API loading | Indeterminate | Manage expectations |
| Multi-step flow | Stepped | Show overall progress |
