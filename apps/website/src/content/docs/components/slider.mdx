## Keyboard Navigation

| Key | Action |
|-----|--------|
| `ArrowRight` / `ArrowUp` | Increase value by step |
| `ArrowLeft` / `ArrowDown` | Decrease value by step |
| `PageUp` | Increase value by 10x step |
| `PageDown` | Decrease value by 10x step |
| `Home` | Set to minimum |
| `End` | Set to maximum |

## Accessibility

- Uses proper ARIA slider role
- Keyboard navigation with arrow keys
- Value announcements for screen readers
- Focus ring for keyboard users
