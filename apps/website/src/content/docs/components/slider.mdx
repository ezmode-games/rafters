## Usage Guidelines

| DO | NEVER |
|:---|:------|
| Provide clear min/max labels and current value display | Use sliders for binary choices (use toggle instead) |
| Use appropriate step sizes for the value range | Create overly sensitive sliders with too many steps |
| Include visual feedback during interaction | Hide the current value from users |
| Offer alternative input methods for precise values | Use sliders when exact values are critical |

## Accessibility Features

- **WCAG AAA Compliance:** Keyboard navigation with arrow keys and page up/down
- **Screen Reader Support:** Value announcements and range descriptions
- **Touch Targets:** Large thumb areas for mobile interaction
- **Alternative Input:** Text input option for precise value entry
