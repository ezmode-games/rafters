# Component Library

Rafters provides shadcn-compatible React components built on framework-agnostic vanilla primitives. Every component works as a drop-in replacement for shadcn/ui while using our own accessible primitives instead of Radix.

## Installation

Add components to your project using the CLI:

```bash
npx rafters add button
npx rafters add dialog
npx rafters add select
```

## Registry API

```bash
# Get all components
GET /registry/index.json

# Get a specific component
GET /registry/components/button.json

# Get primitives
GET /registry/primitives/focus-trap.json

# Initialize a new project
GET /registry/init.json
```
