## Keyboard Navigation

| Key | Action |
|-----|--------|
| `Space` / `Enter` | Open select / select option |
| `ArrowDown` | Open select / move to next option |
| `ArrowUp` | Move to previous option |
| `Home` | Move to first option |
| `End` | Move to last option |
| `Escape` | Close select |
| `A-Z` | Typeahead to matching option |

## Accessibility

- Full ARIA combobox pattern implementation
- Screen reader announcements for options
- Typeahead search for quick navigation
- Proper focus management
