# MCP Server Foundation

**Transform AI Design Capabilities with Live Design Intelligence**

The Rafters MCP server gives AI agents direct access to embedded design reasoning, enabling them to make informed UX decisions instead of guessing. Think of it as a design expert that AI can consult in real-time.

## Why This Matters

Instead of AI agents making arbitrary design choices, they can now:

- **Ask "Why?"** - Access the reasoning behind every design decision
- **Validate Choices** - Check if color combinations meet accessibility standards
- **Manage Complexity** - Calculate cognitive load before overwhelming users
- **Build Trust** - Apply appropriate patterns for different consequence levels
- **Stay Consistent** - Follow systematic design intelligence throughout

## Quick Start Guide

### CLI Integration

The MCP server is built into the Rafters CLI for direct AI agent access:

<div className="bg-gray-50 border border-gray-200 rounded-lg p-4 font-mono text-sm my-6">
  <span className="text-gray-600">$</span> <span className="text-blue-600">npx rafters@latest mcp</span><br/>
  <span className="text-green-600">✓ MCP Server ready for AI agent connection</span><br/>
  <span className="text-gray-500">✓ Design intelligence available via CLI</span>
</div>

**Note**: The dev server (`pnpm dev`) is for component authors only. The MCP server runs separately as part of the CLI for AI agent integration.

### Connect Your AI Agent

For Claude Desktop, add this configuration:

      "command": "npx",
      "args": ["rafters@latest", "mcp"]
    }
  }
}`}</pre>

AI agents making design decisions based on:
- Embedded human design expertise
- Systematic color psychology and accessibility
- Cognitive load budgets and complexity management
- Trust-building patterns for different contexts
