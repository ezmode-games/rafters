# Button

The Button component handles user actions with clear visual feedback.

## When to use each variant

- **default** - Standard actions that don't require emphasis (e.g., Cancel, Close)
- **primary** - Main call-to-action buttons (e.g., Submit, Save, Continue)
- **destructive** - Dangerous or irreversible actions (e.g., Delete, Remove)

## Variants

```tsx live
<Button variant="default">Default</Button>
```

```tsx live
<Button variant="primary">Primary</Button>
```

```tsx live
<Button variant="destructive">Destructive</Button>
```

## Sizes

```tsx live
<Button size="sm">Small</Button>
```

```tsx live
<Button size="default">Default</Button>
```

```tsx live
<Button size="lg">Large</Button>
```

## States

### Disabled

```tsx live
<Button disabled>Disabled</Button>
```
