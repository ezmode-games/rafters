# MDX Layout System Demonstration

This page demonstrates the automatic layout system for MDX documentation. Notice how all content is properly centered and sized without any manual `mx-auto` or max-width classes.

## Automatic Typography & Centering

This paragraph is automatically centered with `max-w-prose mx-auto` for optimal reading width. The system handles all layout concerns, allowing you to focus on content.

### Subsection with Perfect Spacing

The heading above is automatically styled with proper spacing and centering. No manual classes needed.

Here's what the system provides automatically:

- Consistent spacing using phi-based design tokens
- Proper visual hierarchy through typography
- Automatic horizontal centering with mx-auto
