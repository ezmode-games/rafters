# Design Intelligence (Short Reference)

Essential patterns for AI agents using Rafters components:

## 6-Step Integration Process:
1. **Attention**: 1 primary action, 2 secondary max, clear hierarchy
2. **Cognition**: 7±2 items per section, progressive disclosure, consistent patterns
3. **Space**: More space = more important, mathematical relationships, breathing room
4. **Typography**: 6-level hierarchy, scannable structure, WCAG AAA contrast
5. **Enhancement**: Core works without JS, layer improvements, graceful degradation
6. **Trust**: Match friction to consequence (low/medium/high/critical)

## Component Usage Pattern:
```tsx
// Always check component intelligence first:
/**
 * Component Intelligence: cognitiveLoad=X, trustLevel=Y
 * Use for: [specific contexts]
 * Requires: [necessary patterns]
 */

// Apply integration checklist:
// □ Attention hierarchy clear
// □ Cognitive load appropriate  
// □ Spatial relationships support intent
// □ Typography creates scannable flow
// □ Progressive enhancement planned
// □ Trust level matches consequence
```

## Quick Examples:

**Landing Page**: Single primary CTA, supporting alternatives, clear hierarchy
**Forms**: Group related fields, 7±2 per section, progressive steps  
**Dashboards**: Data hierarchy (value > label > context), scannable metrics
**Dialogs**: Trust level matches consequence, progressive confirmation for critical actions

**Success Criteria**: Users immediately understand what's important, mental effort feels justified, relationships are clear, information flows naturally, core always works, confidence builds appropriately.
